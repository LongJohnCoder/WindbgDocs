Running ADPlus for the First Time

To run ADPlus, open a Command Prompt window, move to the folder where you installed or copied the debuggers, and type adplus.vbs.

You might be prompted to change your default script interpreter from WScript.exe to CScript.exe. We strongly recommend that you enable ADPlus to set CScript as the default script interpreter.
