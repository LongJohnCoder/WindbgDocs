Debugging Tools for Windows

/basevideo

The /basevideo parameter directs Windows to use VGA video mode.

/basevideo 

Example

multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /fastdetect /basevideo

Bootcfg Command

bootcfg /addsw BV /ID 1

Comments

The /basevideo parameter is supported only on Windows Server 2003, Windows XP, and Windows 2000. On Windows Vista and later, set the vga parameter using BCDEdit, see BCDEdit /set.
