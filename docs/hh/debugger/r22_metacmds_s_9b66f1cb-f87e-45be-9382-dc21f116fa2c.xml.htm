.sympath (Set Symbol Path)

The .sympath command changes the default path of the host debugger for symbol search.

Syntax

.sympath[+] [Path [; ...]] 

Parameters

+
Specifies that the new directories will be appended to (rather than replace) the previous symbol search path.
Path
The fully qualified path, on the host computer, of a new symbol search. Multiple symbol paths should be separated with semicolons. If Path is omitted, the current symbol path is displayed.

Environment

Modes user mode, kernel mode
Targets live, crash dump
Platforms all

Comments

New symbol information will not be loaded when the symbol path is changed. You can use the .reload (Reload Module) command to reload symbols if deferred symbol loading is off. See Deferred Symbol Loading for details.

Additional Information

For details and other ways to change this path, see Symbol Path.
