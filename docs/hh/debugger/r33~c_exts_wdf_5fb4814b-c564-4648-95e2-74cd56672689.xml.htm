!wdfkd.wdfgetdriver

The !wdfkd.wdfgetdriver extension displays the name of the current default driver.

Syntax

!wdfkd.wdfgetdriver 

DLL

Wdfkd.dll

Additional Information

For more information, see Kernel-Mode Driver Framework Debugging.
