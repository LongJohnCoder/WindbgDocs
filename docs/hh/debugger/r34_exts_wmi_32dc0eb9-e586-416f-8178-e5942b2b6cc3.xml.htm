SieExtPub.dll

The SieExtPub.dll extension module is not part of Microsoft Debugging Tools for Windows. To get extensions from SieExtPub.dll (for example, !critlist), download the Debug Diagnostic Tool from the  Microsoft Download Center. At the download center, search for "Debug Diagnostic Tool".
